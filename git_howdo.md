# Инструкция по git

## Создание репозитория
``` sh
 git init
```

## Добавление файла
``` sh
git add
```

## Фиксация версии с комментарием
``` sh
git commit -m "Message"
```

## Просмотр всех версий
``` sh
git log
```

## Краткий просмотр всех версий
``` sh
git log oneline
```

## Выбор версии
``` sh
git checkout
```
